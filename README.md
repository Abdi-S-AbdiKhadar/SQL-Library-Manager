
# SQL-Library-Manager

## Project Description

An application for local library to help them manage their collection of books. The librarian has been using a simple SQLite database and has been entering data manually. This application will provide a more intuitive way to manage the library's collection of books.

## Tools Used:
    
- initialized the basic structure for your Express project
    -> npx express-generator --view=pug

- installed npm 
    -> npm install

- installed sequelize 
    -> npm install sequelize
    
- installed sqlite3   
    -> npm install qlite3

- added the sequelize-cli package as a dev dependency
    ->  npm install --save-dev sequelize-cli  
    we'll be using this to create our models.

- intialized sequelize to create basic database set up (later cusomized)
    -> npx sequelize init
